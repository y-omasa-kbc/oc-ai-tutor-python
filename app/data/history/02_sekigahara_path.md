# 関ヶ原への道 ― 豊臣政権の黄昏と徳川の覇権

天下統一を成し遂げた秀吉であったが、その晩年は対外侵略と後継者問題により、政権の基盤が大きく揺らぐこととなった。この中編では、秀吉晩年の朝鮮出兵から、関ヶ原の戦いにおける徳川家康の勝利までを取り上げる。

## 第3章 豊臣政権の黄昏と内部抗争

### 文禄・慶長の役とその代償

1592年(文禄元年)から始まった朝鮮出兵(文禄・慶長の役)は、秀吉の過大な領土欲と、統一後の国内武力の矛先を国外へ向ける意図が含まれていた。しかし、この戦争は明の介入や朝鮮民衆の抵抗、そして補給線の伸び切りにより泥沼化し、参加した大名たちに甚大な軍費と兵力の消耗を強いた。

この出兵は、豊臣政権内部に深刻な亀裂を生じさせた。前線で苦闘した加藤清正や福島正則ら「武断派」と、後方で兵站や行政を担い、戦況を冷静に報告した石田三成ら「文治派(吏僚派)」の対立である。この対立は感情的な憎悪を伴うものとなり、秀吉の死後に政権を崩壊させる大きな要因となった。

### 五大老・五奉行制の導入

1598年(慶長3年)、自身の死を予感した秀吉は、幼少の嫡男・秀頼を支えるための集団指導体制を構築した。これが「五大老・五奉行」である。

- **五大老**: 徳川家康、前田利家、宇喜多秀家、毛利輝元、上杉景勝。有力大名による政治的決定機関。
- **五奉行**: 浅野長政、石田三成、増田長盛、長束正家、前田玄以。実務を担う官僚グループ。

秀吉はこの体制により権力の均衡を図ったが、その死とともに、筆頭大老であった徳川家康が公然と自らの覇権を追求し始めたことで、均衡はあっけなく崩れた。

### 七将襲撃事件と徳川家康の台頭

慶長4年(1599年)、両派の調停役であった前田利家が病没すると、武断派の怒りは石田三成へと爆発した。福島正則、加藤清正、黒田長政ら「七将」が三成の屋敷を襲撃する事件が発生した。三成は佐和山城へ隠居を余儀なくされ、この紛争を調停した家康が、豊臣政権の実権を事実上掌握することとなった。

## 第4章 関ヶ原の戦いと徳川の覇権

家康の独走に対し、石田三成は上杉景勝らと連携して反旗を翻した。これが1600年(慶長5年)の「関ヶ原の戦い」である。

### 天下分け目の決戦と戦後処理

関ヶ原の戦いは、東軍(徳川方)と西軍(石田方)に日本全土の大名が二分される大規模な内戦となった。しかし、小早川秀秋の寝返りなどもあり、わずか一日で東軍の勝利が決定した。家康はこの勝利を受けて、西軍に加担した大名たちに対して徹底的な処分を行った。

| 処分の種類 | 概要 | 具体的な影響 |
| :---- | :---- | :---- |
| 改易 | 領地を完全に没収。石田三成、宇喜多秀家ら。 | 西軍の中核勢力の完全排除と、徳川直轄領(天領)の拡大。 |
| 減封 | 領地を大幅に削減。毛利輝元、上杉景勝ら。 | 毛利家は120万石から約30万石へ削減。影響力を大幅に低下させた。 |
| 移封(転封) | 領地を別の場所に変更。福島正則ら。 | 東軍功労者を要所に配置し、親藩・譜代による包囲網を構築。 |

この大規模な領地再編により、徳川家は全国の約3分の1を直轄領、または親藩・譜代大名の所領として確保し、圧倒的な経済的・軍事的優位を確立した。
